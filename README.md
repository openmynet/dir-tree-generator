# dir-tree-generator

快速生成文件目录树的vscode插件

```js
  js
  ├─ css              // css
  │  ├─ common.css    // common.css
  │  └─ details.css   
  ├─ images           
  │  └─ logo.png      //
  └─ js               
     ├─ lib           
     │  └─ jquery.js  //
     ├─ common.js     
     ├─ home.js       
     └─ pages.js      //
```

## Features

* 使用 .gitignore 与 .treeignore 进行忽略
* 右键点击生成并复制到剪贴板
